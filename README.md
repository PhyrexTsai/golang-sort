# golang-sort algorithms

### Algorithm list & test

- Bubble sort
- Insertion sort
- Merge sort
- Quick sort
- Selection sort


### How to build package

```
go install golang-sort
```

### How to run test

```
go test -v
```

### Get library

```
go get github.com/PhyrexTsai/golang-sort
```

### How to import

```
import (
	"github.com/PhyrexTsai/golang-sort"
)
```
